# Vericador de links em arquivos markdown

O objetivo deste projeto é desenvolver uma ferramenta poderosa para realizar verificações em massa de links presentes em arquivos markdown, a fim de determinar se estão funcionais ou quebrados. Com a crescente popularidade dos arquivos markdown como uma forma eficiente de criar e compartilhar conteúdo, torna-se essencial garantir que os links neles contidos estejam atualizados e acessíveis.

O Verificador de Links permitirá que os usuários forneçam um arquivo ou um diretório com vários arquivos markdown como entrada. Em seguida, a ferramenta analisará esses arquivos em busca de links e realizará uma verificação automatizada para determinar seu status. Isso incluirá a verificação do código de resposta HTTP de cada link e a análise de sua validade.

Além disso, o projeto também se concentrará em fornecer uma interface amigável e resultados claros para os usuários. Os links serão classificados como funcionais ou quebrados, e um relatório detalhado será gerado, destacando os links problemáticos e fornecendo informações relevantes, como o status de cada link e sua localização nos arquivos markdown.

Algumas das principais funcionalidades planejadas para o Verificador de Links em Arquivos Markdown incluem:

* Análise de arquivos markdown para identificar todos os links contidos no arquivo.
* Verificação automatizada de links para determinar se estão funcionais ou quebrados.
* Classificação e geração de relatório com o status de cada link.

Com este projeto, os usuários poderão garantir a qualidade e integridade dos links em seus arquivos markdown, economizando tempo e esforço na verificação manual. Além disso, essa ferramenta pode ser uma adição valiosa para equipes de desenvolvimento, escritores técnicos e blogueiros, que desejam manter seus conteúdos atualizados e livres de links quebrados.
